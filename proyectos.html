  • El Ahorcado

    Juego clásico desarrollado en HTML, CSS y JavaScript. Ideal para practicar lógica, arrays y control de flujo.

    • Título: El Ahorcado
    • Versión: 1.0
    • Fecha: Junio 2025
    • Autor: Marcos CM (Estudiante DAM)
    • Descripción: Juego en consola donde el jugador intenta adivinar una palabra. Desarrollado para practicar lógica condicional y estructuras repetitivas.
    • Requisitos del sistema: Java 1.8
    • Lenguaje: Java
    • Funcionalidades:
      • Seleccion aleatoria de palabras
      • Validación de entrada letra a letra
      • Conteo de errores y vidas
    • Interfaz de usuario: Aplicación de terminal, sin entorno gráfico
    • Instrucciones de uso:
      • Compilar con: javac Ahorcado.java
      • Ejecutar con: java Ahorcado
    • Errores comunes y soluciones: - Asegurarse de tener Java instalado. Si no responde, reiniciar consola.
    • Licencia: Proyecto educativo de uso libre para fines de aprendizaje. No apto para distribución comercial sin permiso.
