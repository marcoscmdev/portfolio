# Mi Portfolio – En construcción 🚧

Hola, soy Marcos. Actualmente estudio DAM y me estoy preparando para ser desarrollador de apps multiplataforma (Web, iOS, Android, Escritorio u otras plataformas emergentes) y videojuegos (¡ojalá!).  
Este repositorio es el lugar donde quiero construir un portfolio útil y sincero.  
Una forma de documentar mi camino y compartir lo que aprendo.

## 👀 Qué encontrarás aquí

Este portfolio está dividido en varias secciones principales:

- **Sobre mí**: Una especie de currículum dinámico donde enseño mi formación, mi experiencia y lo que me motiva como desarrollador.

- **Proyectos**: Mis primeros proyectos, algunos cutres, otros no tanto (pero todos reales). Están organizados en tres categorías:

  - **Independientes**: Ideas propias, pequeños experimentos, juegos y apps que he hecho por mi cuenta para aprender, probar o simplemente por curiosidad.

  - **Desde el aula**: Prácticas y proyectos realizados como parte del ciclo DAM. Aunque parten de ejercicios académicos, intento darles un toque personal y profesional.

  - **En equipo**: Proyectos colaborativos. Aquí he aprendido no solo a programar, sino también a comunicarme, repartir tareas y convivir con Git sin dramas.

- **Blog**: Aquí compartiré reflexiones, trucos, tutoriales y avances, a modo de diario de desarrollo. Me gustaría incluir comentarios para poder debatir y recibir feedback.

## 📌 ¿Por qué este portfolio?

Porque quiero trabajar como desarrollador. Ese es mi objetivo principal y por eso estoy metido en todo esto.

Porque me encantaría trabajar en remoto para una empresa internacional, aportando valor desde cualquier sitio y haciendo lo que me gusta.

Porque esto no va solo de tener un "repositorio bonito", sino de documentar mi evolución, mostrar lo que sé hacer y lo que aún estoy aprendiendo.

## 🛠️ Cosas que uso

- Swift y SwiftUI + Android Studio para apps móviles.
- Java, PHP, JavaScript y Python como lenguajes en continuo aprendizaje.
- Git + GitHub para control de versiones.
- Markdown para apuntes y documentación.
  
## 🚀 Mirando al futuro

Este portfolio está abierto a crecer junto a mis habilidades, inquietudes y curiosidad.  
Me interesa especialmente cómo integrar Inteligencia Artificial en el desarrollo de software, explorar tecnologías empresariales como AWS y mejorar mis conocimientos en herramientas y buenas prácticas profesionales (como testing, integración continua o arquitectura de software).

🛠️ Este portfolio estará en constante cambio y evolución, igual que yo como profesional.  
No es un escaparate estático, sino una ventana viva donde muestro lo que aprendo y el camino a dónde me dirijo.
